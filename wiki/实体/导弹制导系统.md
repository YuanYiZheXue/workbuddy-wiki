# 导弹制导系统

> 类型：系统  
> 标签：#控制论 #军事 #航空航天  

---

## 定义

**导弹制导系统**（Missile Guidance System）是控制导弹飞向目标的综合系统，包括**制导律**（Guidance Law）和**控制系统**（Control System）两部分。

---

## 系统组成

### 1. 制导系统（Guidance）
- **功能**：计算期望飞行路径
- **输入**：目标信息、导弹状态
- **输出**：期望加速度/姿态角

### 2. 控制系统（Control）
- **功能**：执行制导指令，控制导弹姿态
- **执行机构**：舵面、推力矢量、燃气舵
- **控制律**：PID、滑模控制、自适应控制

### 3. 传感器系统
- **雷达制导**：主动/半主动/被动
- **红外制导**：热成像、点源红外
- **惯性制导**：IMU + GNSS组合导航
- **地形匹配**：DSMAC、TERCOM

---

## 制导律分类

### 经典制导律
| 制导律 | 原理 | 应用 |
|--------|------|------|
| **比例导引法**（PN） | 加速度 ∝ 视线角速度 | 空空导弹、地空导弹 |
| **纯追踪法** | 速度向量指向目标 | 早期导弹 |
| **平行接近法** | 视线角速度 = 0 | 理论最优 |

### 现代制导律
| 制导律 | 特点 | 应用 |
|--------|------|------|
| **增广比例导引**（APN） | 考虑目标机动 | 现代空空导弹 |
| **最优制导律** | 最小控制能量 | 弹道导弹 |
| **滑模制导律** | 鲁棒性强 | 高机动目标 |
| **微分几何制导** | 考虑空间曲率 | 空天导弹 |

## 控制论在工程中的应用

### 1. 状态空间法
导弹运动状态：
```
X = [位置, 速度, 姿态角, 角速度]ᵀ
```
状态方程：
```
dX/dt = A X + B u + w(t)
```

### 2. 稳定性分析
- **自动驾驶仪稳定性**：保证导弹不发散
- **制导回路稳定性**：制导律 + 控制系统联合分析

### 3. 最优控制
- **中制导**：能量最优路径规划
- **末制导**：脱靶量最小化

## 与金融交易系统的映射

### 系统同构性
| 导弹制导系统 | 金融交易系统 |
|--------------|----------------|
| 目标位置 | 目标仓位/价格 |
| 制导律 | 交易策略 |
| 控制系统 | 执行系统 |
| 传感器噪声 | 市场噪声 |
| 脱靶量 | 跟踪误差 |

### 制导律的金融映射
- **比例导引法** → 均值回归策略（价格偏离 → 反向操作）
- **最优制导律** → 最优执行策略（最小冲击成本）
- **滑模控制** → 鲁棒交易策略（应对市场制度切换）

## Python实现（简化制导律仿真）

```python
import numpy as np
import matplotlib.pyplot as plt

def proportional_navigation():
    """比例导引法仿真"""
    # 初始条件
    missile_pos = np.array([0, 0])    # 导弹位置
    target_pos = np.array([1000, 5000])  # 目标位置
    
    missile_vel = np.array([200, 50])   # 导弹速度
    target_vel = np.array([-10, 50])   # 目标速度
    
    N = 3.0  # 导航常数
    dt = 0.01
    
    trajectory = [missile_pos.copy()]
    
    for t in range(1000):
        # 相对位置、速度
        r = target_pos - missile_pos
        v = target_vel - missile_pos
        
        # 视线角、角速度
        theta = np.arctan2(r[1], r[0])
        # 简化的角速度计算
        omega = np.cross(r, v) / np.dot(r, r)
        
        # 指令加速度（比例导引）
        a_cmd = N * np.cross(omega, missile_vel)
        
        # 更新状态
        missile_vel += a_cmd * dt
        missile_pos += missile_vel * dt
        
        target_pos += target_vel * dt
        
        trajectory.append(missile_pos.copy())
        
        # 判断是否命中
        if np.linalg.norm(r) < 10:
            print(f"命中！时间：{t*dt:.2f}s")
            break
    
    trajectory = np.array(trajectory)
    plt.plot(trajectory[:, 0], trajectory[:, 1])
    plt.xlabel('X (m)')
    plt.ylabel('Y (m)')
    plt.title('Missile Trajectory (Proportional Navigation)')
    plt.grid()
    plt.axis('equal')
    plt.show()

# 运行仿真
proportional_navigation()
```
