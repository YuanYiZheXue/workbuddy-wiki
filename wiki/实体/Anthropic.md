# Anthropic

> AI 安全研究公司，Claude 大模型的开发方。

## 简介

Anthropic 是一家致力于 AI 安全研究的公司，由 Dario Amodei 和 Daniela Amodei 于 2021 年创立（前 OpenAI 高管）。主力产品为 Claude 系列大语言模型。

## 与 Wiki 主题的关系

- 发表了 **Effective harnesses for long-running agents**（Justin Young），提出双 Agent 架构解决长运行 Agent 的上下文管理问题
- Claude Agent SDK 是 Harness 工程的重要实践平台
- Claude 本身是 Agent Harness 理论中的主要模型实例

## 关键产品

- **Claude** — 大语言模型系列（Opus/Sonnet/Haiku）
- **Claude Agent SDK** — 用于构建长运行 Agent 的开发框架
- **Claude Code** — 代码编辑 Agent（在 What Harness Engineering Actually Means 中被提及）
