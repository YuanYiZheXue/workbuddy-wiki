# Claude Agent SDK

Anthropic 提供的 Agent SDK，支持自动 compaction（上下文压缩），减少上下文焦虑问题。

## 基本信息

- **开发方**：Anthropic
- **功能**：Agent 开发 SDK，提供上下文管理、自动 compaction 等功能
- **相关模型**：Claude Opus 4.6（基本消除了上下文焦虑行为）
