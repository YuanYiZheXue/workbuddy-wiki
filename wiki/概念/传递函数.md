# 传递函数

---

## 定义

**传递函数**（Transfer Function）是线性系统在零初始条件下，输出量的拉普拉斯变换与输入量的拉普拉斯变换之比。它描述了系统对输入信号的**频率响应特性**。

数学定义：
```
G(s) = Y(s) / U(s)
```
其中：
- `Y(s)` = 输出量的拉普拉斯变换
- `U(s)` = 输入量的拉普拉斯变换
- `s` = 复频率变量

---

## 核心性质

### 1. 物理意义
传递函数描述了系统的**固有特性**，与外部输入信号无关。

### 2. 典型形式
对于n阶线性系统：
```
G(s) = (b_m s^m + ... + b_1 s + b_0) / (a_n s^n + ... + a_1 s + a_0)
```

### 3. 频率响应
令 `s = jω`，得到频率响应函数：
```
G(jω) = A(ω) e^(jφ(ω))
```
- `A(ω)` = 幅频特性（增益）
- `φ(ω)` = 相频特性（相位延迟）

---

## 典型环节的传递函数

| 环节类型 | 传递函数 | 物理意义 |
|---------|-------------|----------|
| 比例环节 | K | 放大器、传感器 |
| 积分环节 | 1 / (Ts) | 储能元件（电容、弹簧） |
| 微分环节 | Ts | 预测元件 |
| 一阶惯性 | K / (Ts + 1) | RC电路、温度计 |
| 二阶振荡 | K / (T²s² + 2ζTs + 1) | 质量-弹簧-阻尼系统 |
| 时滞环节 | e^(-τs) |  transport delay |

---

## 在控制系统中的应用

### 稳定性判据
通过传递函数的极点位置判断系统稳定性：
- 所有极点都在左半平面 → 稳定
- 有极点在右半平面 → 不稳定
- 极点在虚轴上 → 临界稳定

**劳斯-赫尔维茨判据**：不用求根，直接判断稳定性。

### 品质指标
1. **稳态误差** — 系统趋于稳态后的跟踪误差
2. **超调量** — 过渡过程中最大偏差
3. **调节时间** — 系统进入并保持在误差带内的时间

## 金融市场的映射

### 市场冲击模型
大额订单对价格的影响可以用传递函数描述：

```python
# market_transfer_function.py 核心逻辑
# 简化模型：订单冲击 → 价格影响
# G(s) = K / (T*s + 1) 一阶系统

K = 0.5   # 市场深度倒数（冲击系数）
T = 10    # 价格调整时间常数

# 阶跃响应：大单冲击后的价格调整路径
# P(t) = P0 + ΔP * (1 - e^(-t/T))
```

### 高频交易中的传递函数
- **输入**：订单流（Order Flow）
- **输出**：价格变动
- **传递函数**：描述市场微观结构的信息传递机制

## Python实现

```python
# 使用control库分析传递函数
import control
import matplotlib.pyplot as plt

# 创建传递函数 G(s) = 1 / (10s + 1)
G = control.TransferFunction([1], [10, 1])

# 绘制阶跃响应
t, y = control.step_response(G)
plt.plot(t, y)
plt.xlabel('Time (s)')
plt.ylabel('Response')
plt.title('Step Response of G(s) = 1/(10s+1)')
plt.grid()
plt.show()

# 绘制波特图（幅频/相频特性）
control.bode(G)
plt.show()
```
