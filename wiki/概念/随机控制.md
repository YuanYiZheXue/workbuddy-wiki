# 随机控制

---

## 定义

**随机控制**（Stochastic Control）是研究在**随机扰动**和**噪声测量**下的最优控制策略。它与确定性控制的最大区别是：系统方程包含**随机过程**（如维纳过程、泊松过程）。

---

## 数学模型

### 随机微分方程（SDE）
```
dX(t) = f(X(t), u(t), t) dt + G(X(t), u(t), t) dW(t)
```
其中：
- `W(t)` = 维纳过程（Brown运动）
- `dW(t)` = 高斯白噪声的积分

### 离散时间随机系统
```
X(k+1) = f(X(k), u(k), k) + w(k)
Y(k) = h(X(k), u(k), k) + v(k)
```
其中：
- `w(k)` ~ N(0, Q) ：过程噪声
- `v(k)` ~ N(0, R) ：测量噪声

---

## 核心问题

### 1. 状态估计（滤波）
从噪声测量中估计系统状态。
- **卡尔曼滤波**（线性系统）
- **扩展卡尔曼滤波**（EKF，非线性系统）
- **粒子滤波**（非高斯噪声）

### 2. 最优控制
在噪声环境下，找到最优控制律 `u*(t)`。
- **线性二次高斯（LQG）**问题
- **随机动态规划**（HJB方程）
- **模型预测控制（MPC）**（处理约束）

### 3. 参数辨识
从噪声数据中辨识系统参数。
- **极大似然估计**（MLE）
- **递推最小二乘**（RLS）
- **贝叶斯辨识**

---

## 线性二次高斯（LQG）问题

### 问题设定
- 系统：线性随机系统
- 性能指标：二次型期望 `min E[J]`
  ```
  J = ∫ (xᵀQx + uᵀRu) dt
  ```
- 噪声：高斯白噪声

### 分离原理（Separation Principle）
**状态估计**和**最优控制**可以分离设计！
1. 先设计**卡尔曼滤波器**（状态估计）
2. 再设计**LQR控制器**（确定性等价控制器）
3. 将估计状态 `x̂` 当作真实状态使用

### LQG vs. LQR
| 特性 | LQR | LQG |
|------|------|------|
| 系统模型 | 确定性 | 随机噪声 |
| 状态信息 | 完全已知 | 含噪声测量 |
| 控制器结构 | 状态反馈 | 估计状态反馈 |
| 鲁棒性 | 好 | 可能差（需要鲁棒LQG） |

## 金融系统中的随机控制

### 1. 期权定价（Black-Scholes）
Black-Scholes方程本质上是一个**随机控制问题**：
- 标的资产遵循几何布朗运动（SDE）
- 目标：对冲组合的风险最小化
- 解：Black-Scholes PDE

### 2. 投资组合优化（Merton问题）
连续时间投资组合优化：
- **状态变量**：财富 `W(t)`
- **控制变量**：资产配置比例 `π(t)`
- **目标**：最大化效用函数的期望
- **解**：HJB方程 → 最优投资策略

### 3. 算法交易（Almgren-Chriss模型）
最优执行策略：
- **控变量**：执行速率 `v(t)`
- **扰动**：市场冲击 + 价格噪声
- **目标**：min E[执行成本 + 风险惩罚]
- **解**：随机控制 → 确定性等价问题

## Python实现

```python
import numpy as np
import control
from scipy import linalg

def lqg_design():
    """LQG控制器设计"""
    # 系统参数（倒立摆为例）
    A = np.array([[0, 1], [1, 0]])  # 状态矩阵
    B = np.array([[0], [1]])           # 输入矩阵
    C = np.array([[1, 0]])            # 输出矩阵
    D = np.array([[0]])
    
    # LQR权重
    Q = np.diag([10, 1])  # 状态权重
    R = np.array([[1]])     # 控制权重
    
    # 求解Riccati方程（LQR）
    P = linalg.solve_continuous_are(A, B, Q, R)
    K = np.linalg.inv(R) @ B.T @ P  # 状态反馈增益
    
    # 卡尔曼滤波器设计
    Qn = np.eye(2) * 0.1  # 过程噪声协方差
    Rn = np.array([[1]]) * 0.1  # 测量噪声协方差
    
    # 求解Riccati方程（卡尔曼滤波）
    S = linalg.solve_continuous_are(A.T, C.T, Qn, Rn)
    L = S @ C.T @ np.linalg.inv(Rn)  # 卡尔曼增益
    
    print("LQR反馈增益 K:", K)
    print("卡尔曼增益 L:", L)
    
    return K, L
```
