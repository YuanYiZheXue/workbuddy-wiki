# 工程控制论

---

## 定义

**工程控制论**（Engineering Cybernetics）是研究控制系统设计与分析的工程学科，由钱学森于1954年创立。它将控制理论的数学方法应用于工程系统，核心问题是：**如何设计一个系统，使其按预定规律运动，并对扰动具有鲁棒性**。

## 与金融交易系统的同构性

### 数学模型同构

| 控制系统 | 金融交易系统 |
|----------|----------------|
| 受控对象（导弹、飞行器） | 价格动态系统 |
| 控制量（舵偏角、推力） | 交易订单（买卖、仓位调整） |
| 受控量（位置、速度） | 持仓、盈亏 |
| 扰动（风、气流） | 市场噪声、黑天鹅 |
| 稳定性 | 策略稳健性（最大回撤可控） |
| 超调量 | 最大回撤幅度 |
| 过渡过程时间 | 策略适应新市场的时间 |
| 积分指标最优 | 最小化冲击成本/最大化Sharpe |

### 设计目标对应

1. **稳定性** → 策略不会爆仓，回撤可控
2. **快速响应** → 及时捕捉市场机会
3. **最小超调** → 避免追涨杀跌导致的回撤
4. **抗干扰** → 在噪声市场中稳定盈利
5. **最优控制** → 在风险约束下最大化收益

## Python 实现（模型实现/目录）

```python
# market_transfer_function.py - 市场机制传递函数
# 对应第3章：输入、输出和传递函数
import control
import numpy as np

def market_transfer_function():
    """创建市场冲击的传递函数模型"""
    # 简化模型：订单冲击 → 价格影响
    # G(s) = K / (T*s + 1) 一阶系统
    K = 0.5  # 市场深度倒数
    T = 10   # 价格调整时间常数
    G = control.TransferFunction([K], [T, 1])
    return G

def step_response(G):
    """阶跃响应 - 模拟大单冲击后的价格调整"""
    t, y = control.step_response(G)
    return t, y
```
