# 市场哲学框架

> 从日本蜡烛图分析传统中提炼出的通用哲学思想，适用于金融交易与更广义的决策场景。

---

## 核心命题

### 1. 位置先于信号（位置哲学）

> "位置、位置、还是位置。" — 格里高里·莫里斯

任何技术信号的价值，**完全取决于它出现的位置**。

- 同样一根锤子线，在下跌趋势底部是强看涨信号，在上升顶部则无意义
- 哲学引申：**语境决定意义**，脱离上下文的真理是空洞的
- 应用：决策前先问「这件事发生在什么位置/阶段/环境？」

---

### 2. 阴阳互根（阴阳哲学）

蜡烛图的阴线/阳线直接来源于中国阴阳哲学，经日本吸收后融入交易思想：

| 对立面 | 市场含义 | 哲学含义 |
|--------|----------|----------|
| 阴 / 阳 | 卖方主导 / 买方主导 | 任何事物都有对立统一的两面 |
| 实体 / 影线 | 确定性 / 犹豫性 | 力量与柔韧并存 |
| 上涨 / 下跌 | 乐观 / 悲观 | 物极必反，循环往复 |

**核心洞察**：市场不是单向的，而是在阴阳转换中寻找平衡。交易者的任务不是「预测方向」，而是「识别阴阳转换的节点」。

---

### 3. 三数法则（酒田战法核心）

本间宗久发现：**「3」是自然与市场的核心节律**。

- 三山（三重顶）→ 三重底
- 三川（三重底）→ 三重顶
- 三空（三个跳空缺口）→ 趋势耗尽
- 三兵（白色三兵）→ 强烈上涨
- 三法（大阳+小实体组+大阳）→ 趋势延续

**哲学含义**：
- 事物发展经过「萌芽 → 发展 → 鼎盛」三个阶段
- 第三阶段往往伴随衰竭（三次跳空后考虑反转）
- **应用**：任何趋势/关系/项目，到了「第三次」要警惕转折点

---

### 4. 关键交易者理论（少数决定论）

> 市场价格由少数关键交易者的行为主导，而非「市场整体」。

- 大多数参与者是跟随者，真正决定价格的是大资金
- 蜡烛图的价值：它可视化了关键交易者的意图（通过实体大小和影线）
- **哲学引申**：任何系统中，少数关键节点的行为决定整体走向（二八定律的变体）

---

### 5. 市场即人心（心理映射）

蜡烛图不是冷冰冰的数据，而是**交易群体心理的可视化**：

| 蜡烛形态 | 心理状态 |
|----------|----------|
| 长下影线（锤子线） | 卖方尝试压低价格，但买方强势反击 → 心理反转 |
| 长上影线（流星线） | 买方尝试推高价格，但卖方压制 → 心理枯竭 |
| 实体极小（十字星） | 买卖力量平衡 → 心理犹豫，方向不明 |

**哲学核心**：**价格 = 人性的集体投射**。理解市场就是理解人性。

---

### 6. 过滤原则（多验证伪）

单一信号不可信，需要「过滤」：

- 用 RSI / %D / MACD 等指标过滤蜡烛图信号
- 用「位置」过滤形态有效性
- **哲学引申**：任何断言都需要「多重验证」。真理不是「一次确认」，而是「多重视角下的稳定结论」。

## 与实践的关系

此框架不是「交易技巧」，而是**认知世界的思维工具**：

1. **做决策时** → 先判断「位置」（上下文）
2. **看对立面时** → 寻找阴阳转换的节点
3. **观察趋势时** → 警惕「第三次」的转折信号
4. **分析群体行为时** → 寻找关键决策者（而非平均值）
5. **得出结论时** → 用多重证据过滤，不轻信单一信号
