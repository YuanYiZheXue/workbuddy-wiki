# 离散控制系统

---

## 定义

**离散控制系统**是指系统中一个或多个变量在时间上是离散的（仅在采样时刻变化）的控制系统。与连续系统不同，离散系统使用**差分方程**描述，分析工具是**Z变换**。

---

## 与连续系统的对比

| 特性 | 连续系统 | 离散系统 |
|------|----------|----------|
| 时间变量 | 连续 t | 离散 kT（T为采样周期） |
| 描述方程 | 微分方程 | 差分方程 |
| 变换工具 | 拉普拉斯变换 | Z变换 |
| 稳定性判据 | 左半平面 | 单位圆内 |
| 频率响应 | 傅里叶变换 | 离散傅里叶变换（DFT） |

---

## Z变换

### 定义
离散序列 `x(kT)` 的Z变换：
```
X(z) = Σ_{k=0}^{∞} x(kT) z^{-k}
```

### 重要性质
1. **时移性质**：`Z{x(k-n)} = z^{-n} X(z)`
2. **初值定理**：`x(0) = lim_{z→∞} X(z)`
3. **终值定理**：`x(∞) = lim_{z→1} (z-1)X(z)` （需满足稳定性）

### 常用Z变换对
| 时域序列 | Z变换 |
|---------|--------|
| δ(k)（单位脉冲） | 1 |
| 1(k)（单位阶跃） | z/(z-1) |
| kT | Tz/(z-1)² |
| a^{kT} | z/(z-a) |
| e^{-αkT} | z/(z-e^{-αT}) |

---

## 离散传递函数

对于离散系统：
```
y(k) + a_1 y(k-1) + ... + a_n y(k-n) = b_0 u(k) + b_1 u(k-1) + ... + b_m u(k-m)
```

Z变换后：
```
Y(z) / U(z) = G(z) = (b_0 + b_1 z^{-1} + ... + b_m z^{-m}) / (1 + a_1 z^{-1} + ... + a_n z^{-n})
```

---

## 稳定性判据

### 双线性变换（Tustin变换）
将z平面映射到w平面：
```
w = (z-1)/(z+1)
```
然后应用连续系统的稳定性判据（劳斯判据）。

### 朱里判据（Jury Criterion）
直接对特征方程 `D(z) = 0` 的系数进行判断，类似劳斯表。

**稳定性条件**：特征方程的所有根都在单位圆内（`|z| < 1`）。

---

## 采样定理（Shannon-Nyquist）

**采样频率** `fs` 必须大于信号最高频率 `fmax` 的两倍：
```
fs > 2 fmax
```
否则会发生**混叠（Aliasing）**现象。

---

## 金融系统中的离散控制

### 1. 高频交易（HFT）
- 决策时刻：离散（每毫秒/微秒）
- 控制量：订单发送（离散事件）
- 分析工具：离散时间马尔可夫链、时序分析

### 2. 投资组合再平衡
- 再平衡周期：每日/每周（离散）
- 控制策略：离散时间动态规划
- 分析工具：离散时间随机控制

### 3. 技术指标计算
- 移动平均、RSI、MACD等技术指标都是离散时间计算
- 采样周期：1分钟、5分钟、日线等

## Python实现

```python
import control
import numpy as np
import matplotlib.pyplot as plt

def discrete_system_analysis():
    """离散控制系统分析"""
    # 创建离散传递函数 G(z) = (z+0.5) / (z^2 - 1.3z + 0.4)
    # 采样周期 T = 0.1s
    G = control.TransferFunction([1, 0.5], [1, -1.3, 0.4], dt=0.1)
    
    # 计算脉冲响应
    t, y = control.impulse_response(G, T=np.arange(0, 10, 0.1))

    t, y = control.step_response(G)
    plt.stem(t, y)  # 离散系统用stem绘图
    plt.xlabel('Time (s)')
    plt.ylabel('Response')
    plt.title('Discrete System Step Response')
    plt.grid()
    plt.show()
    
    return is_stable
```
