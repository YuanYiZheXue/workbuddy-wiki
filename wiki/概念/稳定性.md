# 稳定性

---

## 定义

**稳定性**是控制系统的最基本性质：当系统受到有界扰动时，其输出是否保持有界（BIBO稳定性）。对于线性定常系统，稳定性等价于系统传递函数的所有极点都在左半复平面。

---

## 分类

### 1. 渐近稳定性
扰动消失后，系统状态最终回到平衡位置。
- **条件**：所有特征值都有负实部
- **例子**：弹簧阻尼系统（阻尼比 > 0）

### 2. 李雅普诺夫稳定性
扰动消失后，系统状态保持在平衡位置附近（不一定回到原点）。
- **条件**：存在正定李雅普诺夫函数 V(x)，且其导数负半定
- **意义**：适用于非线性系统的稳定性分析

### 3. 有限时间稳定性
在有限时间区间内，系统状态不超过给定界限。
- **应用**：变系数系统、导弹发射窗口

### 4. 鲁棒稳定性
系统参数在一定范围内变化时，稳定性保持不变。
- **意义**：实际工程系统必须考虑鲁棒性

---

## 稳定性判据

### 劳斯-赫尔维茨判据
不用求解特征方程，直接判断稳定性。

**劳斯表构造**：
```
s^n    : a_n   a_{n-2}   a_{n-4}  ...
s^{n-1}: a_{n-1}   a_{n-3}   a_{n-5}  ...
s^{n-2}: b_1   b_2      b_3     ...
...
```
**判据**：劳斯表第一列所有元素都大于0 → 系统稳定。

### 奈奎斯特判据
通过开环频率响应判断闭环稳定性。
- **原理**：开环传递函数绕(-1, j0)点的圈数 = 右半平面极点数
- **优点**：无需精确数学模型，可通过实验数据判断

### 波特图判据
基于幅频/相频特性判断稳定性。
- **相位裕度**：增益为0dB时，相位与-180°的差值
- **增益裕度**：相位为-180°时，增益与0dB的差值

---

## 非线性系统的稳定性

### 描述函数法
用等效线性化方法分析非线性系统。
- **适用**：典型非线性（饱和、死区、滞环）
- **局限**：只能判断是否存在稳定周期解（自激振荡）

### 相平面法
在二维状态空间中分析系统轨迹。
- **平衡点的类型**：节点、焦点、鞍点、中心
- **极限环**：孤立的周期解（稳定/不稳定）

## 金融系统的稳定性映射

### 策略稳健性
| 控制系统 | 金融交易系统 |
|----------|----------------|
| 受控对象稳定 | 策略在不同市场环境下都能盈利 |
| 控制器参数鲁棒 | 策略参数在小范围变化时性能不恶化 |
| 抗干扰能力 | 策略在异常行情下不会爆仓 |

### 最大回撤 = 超调量
```
最大回撤 = (峰值 - 谷值) / 峰值
```
控制系统中，超调量：
```
σ% = (y_max - y_ss) / y_ss * 100%
```

### 风险管理中的稳定性
1. **仓位管理的稳定性**：固定分数 vs. 凯利公式
2. **组合优化的稳定性**：均值-方差优化 vs. 鲁棒优化
3. **止损机制**：类似于控制系统的约束条件

## Python实现

```python
import control
import numpy as np

def stability_analysis():
    """控制系统稳定性分析"""
    # 创建开环传递函数 G(s) = 1 / (s(s+1)(s+2))
    G = control.TransferFunction([1], [1, 3, 2, 0])
    
    # 计算闭环极点
    poles = control.pole(G)
    print("闭环极点:", poles)
    
    # 判断稳定性
    if all(np.real(p) < 0 for p in poles):
        print("系统稳定")
    else:
        print("系统不稳定")
    
    # 计算相位裕度和增益裕度
    gm, pm, wg, wp = control.margin(G)
    print(f"增益裕度: {gm:.2f} dB")
    print(f"相位裕度: {pm:.2f} 度")
    
    return poles, gm, pm
```
