# 状态空间法

## 定义

状态空间法（State-Space Representation）是一种用一阶微分方程组描述动态系统的方法。与传递函数（频域）不同，状态空间法可以用矩阵形式描述多输入多输出（MIMO）系统，特别适合高阶系统建模。

## 为什么需要从电路跃升到状态空间？

单个电路板内部，二阶系统（如 RLC 电路）是手算的极限。但在导弹、航天器等复杂系统中，需要用状态空间法描述整个飞行器在空间中的姿态（俯仰、偏航、滚转）。系统是由几十甚至上百个一阶、二阶微分方程组成的多维复杂系统。

## 状态空间基本形式

```
连续系统：
  ẋ(t) = A x(t) + B u(t)
  y(t)   = C x(t) + D u(t)

其中：
  x(t)：状态向量（系统内部状态）
  u(t)：输入向量
  y(t)：输出向量
  A：系统矩阵
  B：输入矩阵
  C：输出矩阵
  D：传递矩阵
```

## 与传递函数的对比

| 对比维度 | 传递函数 | 状态空间 |
|----------|----------|----------|
| 表示域 | 频域（s 域） | 时域 |
| 系统类型 | 单输入单输出（SISO） | 多输入多输出（MIMO） |
| 初始条件 | 难以处理 | 自然纳入 |
| 系统阶数 | 隐式 | 显式（状态变量个数 = 阶数） |

## 在嵌入式系统中的应用

- **飞控系统**：描述飞行器六自由度运动
- **机器人控制**：多关节机器人的动力学建模
- **卡尔曼滤波**：状态空间形式是 KF 的基础
- **LQR 控制**：基于状态空间的最优控制设计
