# 分布参数系统

---

## 定义

**分布参数系统**（Distributed Parameter System）是指系统状态需要用**分布函数**（如 `y(x,t)`）描述的系统，其数学模型是**偏微分方程**（PDE）或**积分方程**。与集中参数系统（用常微分方程描述）不同，分布参数系统具有**无限维状态空间**。

---

## 典型例子

### 1. 弹性结构体
- 梁的振动：`∂²y/∂t² + c ∂⁴y/∂x⁴ = u(x,t)`
- 板的振动：更高阶偏微分方程

### 2. 热传导过程
- 一维热传导：`∂T/∂t = α ∂²T/∂x² + Q(x,t)`
- 边界条件：对流、辐射、 insulated

### 3. 流体管道
- 流量-压力动态：`∂p/∂t + c² ∂q/∂x = 0`
- 波方程：描述压力波传播

### 4. 电磁场
- 麦克斯韦方程：一组偏微分方程
- 微波谐振腔、传输线

---

## 数学描述

### 一般形式
```
∂y/∂t = A y + B u
z = C y
```
其中：
- `y(x,t)` 是分布状态（函数）
- `A` 是微分算子（如拉普拉斯算子 ∇²）
- `B` 是输入算子
- `C` 是输出算子

### 边界条件
- **Dirichlet条件**：指定边界上的 `y`
- **Neumann条件**：指定边界上的 `∂y/∂n`
- **Robin条件**：混合条件

---

## 控制系统设计方法

### 1. 模态展开法（Modal Expansion）
- 将偏微分方程的解展开为正交函数基（如傅里叶级数、特征函数）
- 截断为有限维系统（Galerkin方法）
- 对截断后的有限维系统设计控制器

### 2. 直接设计法
- 直接在无穷维空间设计控制器
- 使用**算子理论**、**半群理论**
- 理论上严格，但实现复杂

### 3. 近似方法
- **有限元法**（FEM）
- **有限差分法**（FDM）
- **边界控制**：仅在边界施加控制输入

---

## 稳定性分析

### 特征值问题
分布参数系统的稳定性由算子 `A` 的谱（特征值集合）决定：
- 所有特征值的实部 < 0 → 渐近稳定
- 存在特征值实部 > 0 → 不稳定

### 无穷维系统的特点
- 特征值有**无穷多个**
- 可能有**连续谱**
- 稳定性判据更复杂

---

## 金融系统的映射

### 1. 连续时间定价模型
- **Black-Scholes方程**：偏微分方程
  ```
  ∂V/∂t + (1/2)σ²S² ∂²V/∂S² + rS ∂V/∂S - rV = 0
  ```
- 这是典型的分布参数系统（状态是函数 `V(S,t)`）

### 2. 订单簿动态
- 限价订单簿（LOB）的演化：密度方程
- **连续极限**：将离散订单流建模为连续密度
- 相关文献：Avellaneda & Stoikov (2008)

### 3. 高频做市
- 库存风险管理：连续时间随机控制
- HJB方程（偏微分方程）
- 分布参数系统的随机控制

## Python实现

```python
import numpy as np
import matplotlib.pyplot as plt
from scipy import integrate

def heat_equation_simulation():
    """一维热传导方程仿真（分布参数系统）"""
    # 参数
    L = 1.0       # 杆长度
    T = 0.1       # 热扩散系数
    Nx = 50       # 空间离散点数
    Nt = 1000     # 时间步数
    
    x = np.linspace(0, L, Nx)
    dx = x[1] - x[0]
    dt = 0.001
    
    # 初始条件：杆中心有热源
    T_emperaure = np.zeros(Nx)
    T_emperaure[Nx//2-5:Nx//2+5] = 100  # 中心高温
    
    # 时间演化（显式有限差分）
    for n in range(Nt):
        T_new = T_emperaure.copy()
        for i in range(1, Nx-1):
            T_new[i] = T_emperaure[i] + alpha * dt / dx**2 * (
                T_emperaure[i+1] - 2*T_emperaure[i] + T_emperaure[i-1]
            )
        T_emperaure = T_new
        
        # 每100步记录一次
        if n % 100 == 0:
            plt.plot(x, T_emperaure, label=f't={n*dt:.3f}s')
    
    plt.xlabel('Position (m)')
    plt.ylabel('Temperature (°C)')
    plt.title('Heat Equation (Distributed Parameter System)')
    plt.legend()
    plt.grid()
    plt.show()
```
