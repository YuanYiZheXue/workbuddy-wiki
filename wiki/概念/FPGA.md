# FPGA

## 定义

FPGA（Field-Programmable Gate Array，现场可编程门阵列）是一种可在制造后通过编程重新配置硬件电路的半导体器件。与 ASIC 不同，FPGA 的硬件逻辑可以在部署后修改。

## FPGA vs ASIC

| 对比维度 | FPGA | ASIC |
|----------|------|-------|
| 开发成本 | 低（可重复编程） | 高（掩膜成本） |
| 单位成本 | 高 | 低（大批量） |
| 性能 | 较高 | 最高 |
| 功耗 | 较高 | 低 |
| 灵活性 | 高（可重构） | 无 |
| 开发周期 | 短 | 长 |

## 开发流程

```
设计输入（Verilog/VHDL）
   ↓
功能仿真（ModelSim/VCS）
   ↓
综合（Synthesis）→ 网表
   ↓
布局布线（Place & Route）
   ↓
时序仿真
   ↓
生成比特流（Bitstream）
   ↓
板级调试
```

## 硬件描述语言

| 语言 | 特点 | 适用场景 |
|------|------|----------|
| Verilog | 最流行、语法类似 C | 通用 FPGA 开发 |
| VHDL | 强类型、严谨 | 航空航天、军工 |
| Chisel | 基于 Scala、可参数化 | 高级 CPU 设计 |
| SystemVerilog | Verilog 增强版 | 复杂验证 |
